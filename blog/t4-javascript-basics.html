Explain the difference between accessing data from arrays and objects.

Arrays are sets of data. You count arrays starting from 0. You can access data from arrays using this count. You can also change data in arrays and pop data in and out of arrays.

Objects. You put properties into an object, or I guess you add propoerties to an object. You can change, add and delete properties. Then you access data by calling that property's name, and you can use dots and brackets to chain together properties in order to access nested properties.

Explain what functions are and why they are useful.

Functions do things like running methods. You name a function and then you can invoke a function to run again by calling its name. You can also use parameters in functions to hold space for information later on, and then you can pass information into the function in those placeholders. This is great as it means you can test a whole bunch of scenarios with the same function, by passing different arguments into the placeholders.
