parentheses

Contain parameters passed to functions

brackets

You use square brackets around arrays and to access things in arrays.

braces

Curly brackets declare objects

Enclose blocks of code

single quotes and double quotes

These contain strings. There is no difference between them but if one type is containing a string you can use the other type within that string without escaping the string, ie for grammatical reasons.
