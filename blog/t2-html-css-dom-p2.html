What is grid based design?

Designing your page based on a grid structure, ie dividing it into rows and columns and letting some elements take up several columns (invisibly) to make it look good.

What's all the hype about responsive webpages?

Since 2013 Google penalises webpages that aren't mobile compatible. So all webpages need to 'respond' to the size of th screen by getting clearer and less crowded the smaller a screen gets.

What is semantic structure

That's just using different tags for different sized headings so you can style them all at once in your CSS.
