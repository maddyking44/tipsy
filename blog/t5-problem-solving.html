Functions

  • .map()

    • Maps a function onto every element in an array. So if your array called myArray contains [2, 3, 5], you can myArray.map(x => x * 2), this will times every element in the array by 2, giving you a result of [4, 6, 10].

  • .filter()

    • You can use .filter to filter out elements in an array that don't pass a test, i.e. in an array of words, ['tree', 'present', 'chicken', 'peace', 'pie'], you can words.filter out words with more than 4 letters, [tree, pie].

  • .reduce()

    • Reduces numbers in an array to a single number, i.e. by accumulating them and adding them together. So an array of digits [6, 5, 4] can digits.reduce to 15.
